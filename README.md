# 🤖 PAXTON-MD WhatsApp Bot

Advanced WhatsApp Multi-Device Bot with 150+ commands.

## ✨ Features

- 📱 **General Commands** - Menu, help, ping, info, profile
- 🎵 **Media Downloader** - YouTube, TikTok, Instagram, Facebook
- 🎮 **Playful Commands** - Jokes, facts, roasts, games
- 👥 **Group Management** - Tagall, promote, demote, mute, lock
- 💍 **Marriage System** - Marry, divorce, love meter, ship
- 🛡️ **Anti Features** - Anti-link, anti-spam, anti-toxic
- 🐛 **Bug/Hazard** - Bug reports, hack, hijack
- ⚙️ **Bot Settings** - Auto-status, auto-react, auto-view
- 👑 **Owner Commands** - Ban, unban, broadcast, join, leave

## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- npm or yarn

### Installation

```bash
git clone https://github.com/YOUR_USERNAME/PaxtonMD.git
cd PaxtonMD
npm install
node index.js
