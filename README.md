## 🚀 Quick Start

### Prerequisites
- Node.js 16+
- npm or yarn
- Termux (for Android) or any Linux system

### Installation

```bash
# Clone repository
git clone https://github.com/jarvismd98-afk/Paxton-Md.git
cd Paxton-Md

# Install dependencies
npm install

# Start the bot
node index.js
